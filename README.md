# daliacordeiro.art

Portfolio website for **Dalia Cordeiro**, a Portuguese visual artist working in oil, pastel and mixed media.

Live: https://dalia-cordeiro.pages.dev

## Architecture

```
Browser ──► Cloudflare Pages (static HTML/CSS/JS)
                 │
                 │  build-time fetch (GROQ)
                 ▼
            Sanity CDN (content + images)
                 │
                 │  runtime POST (client-side)
                 ▼
            Web3Forms (contact form)
```

The site is **fully static** (SSG) — no server runtime. Content is fetched from Sanity at build time and baked into HTML. The only client-side JavaScript is for the artwork grid filters, detail modal, and contact form (~8 KB gzip total, plus React runtime).

## Tech Stack

| Layer | Technology | Purpose |
|-------|-----------|---------|
| **Frontend** | [Astro](https://astro.build/) v5 | Static site generator, page routing, i18n |
| **UI Islands** | [React](https://react.dev/) 19 | Interactive components (filters, modal, form) |
| **CMS** | [Sanity](https://www.sanity.io/) v3 | Headless CMS with field-level PT/EN translation |
| **Styling** | Vanilla CSS | Custom editorial design, 3 themes, 4 accents |
| **Fonts** | Newsreader + Inter Tight | Self-hosted WOFF2 (latin + latin-ext subsets) |
| **Hosting** | [Cloudflare Pages](https://pages.cloudflare.com/) | Static deploy, global CDN |
| **Contact** | [Web3Forms](https://web3forms.com/) | Form submission without backend |
| **Monorepo** | [pnpm](https://pnpm.io/) workspaces | `web/` + `studio/` in one repo |

## i18n Strategy

- **Two languages**: Portuguese (default) and English
- **Prefixed routes**: `/pt/...` and `/en/...` with translated path segments
- **Field-level translation** in Sanity (not document-level) — one dataset, `{pt, en}` objects
- **PT fallback**: if an English field is empty, the Portuguese value is shown
- **hreflang**: correct alternates on every page, `x-default` → PT
- **Sitemap**: generated by `@astrojs/sitemap` with i18n config

## Design System

Three themes (`paper` / `bright` / `dark`) and four accent colors (`coral` / `sage` / `lavender` / `mustard`) controlled via CSS custom properties and body classes. Currently hardcoded to `theme-paper accent-coral` — can be made CMS-editable in the future.

## Development

```bash
# Install dependencies
pnpm install

# Run Astro dev server (http://localhost:4321)
pnpm dev:web

# Run Sanity Studio (http://localhost:3333)
pnpm dev:studio

# Build static site
pnpm build:web

# Deploy to Cloudflare Pages
wrangler pages deploy web/dist --project-name dalia-cordeiro
```

### Environment Variables

Copy `.env.example` to `.env` and fill in:

| Variable | Description |
|----------|-------------|
| `SANITY_PROJECT_ID` | Sanity project ID (`gwtbwm5k`) |
| `SANITY_DATASET` | Dataset name (`production`) |
| `SANITY_API_TOKEN` | API token with editor permissions |
| `PUBLIC_WEB3FORMS_KEY` | Web3Forms access key for contact form |

## Migration Script

```bash
node scripts/import-from-data.js
```

Reads artwork data and images from `prototype/`, uploads to Sanity, and creates all documents. Safe to run multiple times (idempotent — patches existing docs instead of duplicating).
